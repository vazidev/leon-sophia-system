# Synthesis — Debate Protocol

## Purpose
Structure the exchange between LEON and SOPHIA so disagreements are productive and converge toward resolution rather than cycling.

## Debate Rules
1. Each agent states positions, not preferences. "The evidence shows X" not "I think X."
2. Challenges must be specific. "This is wrong" is not a valid challenge. "Claim X on line Y fails because Z" is.
3. Concessions are recorded. If an agent concedes a point, it's logged and not re-litigated.
4. New evidence can be introduced at any round. New arguments must be based on that evidence.
5. Repetition without new evidence is not permitted after Round 2.

## Debate Structure

### Round 0: Position Statement
- LEON states the core recommendation and its top 3 supporting claims.
- SOPHIA states her top 3 challenges.

### Round 1: Direct Response
- LEON responds to each SOPHIA challenge specifically.
- SOPHIA responds to each LEON counter-argument specifically.

### Round 2: Concession and Escalation
- Each agent identifies which of the other's points they accept.
- Remaining disagreements are classified as disputes for arbitration.

### Round 3 (if needed): Synthesis Proposal
- Either agent may propose a synthesis position that incorporates elements of both.
- The other agent responds: accept, reject with modification, or reject with escalation.

## Debate Log Format
```
Round: [0 / 1 / 2 / 3]
Agent: [LEON / SOPHIA]
Type: [Position / Challenge / Counter / Concession / Synthesis]
Claim: [the specific claim being made]
Evidence: [supporting evidence or "none"]
Status: [open / conceded / disputed / resolved]
```
