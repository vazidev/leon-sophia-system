# LEON — Tool Creation Protocol

## When to Create a Tool
Create a new tool when:
1. A recurring manual process has been performed 3+ times identically.
2. A capability gap blocks progress on a critical path item.
3. An existing tool produces output that requires consistent manual transformation.

Do NOT create a tool when:
- The process will run fewer than 3 times.
- An existing tool can be configured to meet the need.
- The tool would require more maintenance than the time it saves.
